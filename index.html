Methodology

1. Research Design

This capstone adopts a mixed-methods approach, combining statistical rigor with practical interpretability. The methodology is divided into three stages:

  1. Data Collection and Preprocessing
  2. Application of Visualization Techniques (grounded in mathematical/statistical principles)
  3. Evaluation and Interpretation

This structure ensures that visualization methods are both mathematically valid and practically useful for decision making.

2. Data Sources and Preparation

Data will be obtained from open repositories such as Kaggle and World Bank datasets. The preprocessing steps include:

  • Handling Missing Data: Mean/median imputation or multiple imputation.

  • Normalization:

\[x' = \frac{x - \min(x)}{\max(x) - \min(x)}\]

  • Standardization:

\[z = \frac{x - \mu}{\sigma}\] where \(\mu\) is the mean and \(\sigma\) the standard deviation.

  • Dimensionality Reduction (PCA): Given a data matrix (X), PCA is computed by eigenvalue decomposition of the covariance matrix:

    \[C = \frac{1}{n-1}X^TX\]

    Eigenvectors of (C) provide principal components, and eigenvalues indicate explained variance.

3. Visualization Techniques and Statistical Methods

Visualization techniques will be selected based on data type and analytical goal. Each visualization is tied to a statistical foundation.

3.1 Univariate Visualizations

  • Histogram: A histogram estimates the probability density function using bins. For bin width (h),

    \[\hat{f}(x) = \frac{1}{nh} \sum_{i=1}^{n} K\left(\frac{x - x_i}{h}\right)\] where \(K\) is a kernel (commonly uniform or Gaussian).

  • Boxplot: Represents quartiles. Interquartile range (IQR):

\[\text{IQR} = Q_3 - Q_1\]

3.2 Bivariate and Multivariate Visualizations

  • Scatterplot with Regression Line: The regression line is based on the least squares estimate:

    \[\hat{y} = \beta_0 + \beta_1 x\] with coefficients: \[\beta_1 = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2}\] \[\beta_0 = \bar{y} - \beta_1 \bar{x}\]

  • Correlation Heatmap: Correlation coefficient (Pearson):

\[r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}}\]

3.3 Predictive and Inferential Visualizations

ROC Curve:

True Positive Rate (TPR) and False Positive Rate (FPR): \[TPR = \frac{TP}{TP+FN}, \quad FPR = \frac{FP}{FP+TN}\]

Area Under the Curve (AUC):

\[AUC = \int_0^1 TPR(FPR^{-1}(x)) dx\] Confidence Interval Plot: For mean \(\bar{x}\): \[CI = \bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}\]
